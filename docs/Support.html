PlanTogether Support

Last updated: February 14, 2026

If you need help with PlanTogether, contact us and we’ll get back to you as quickly as possible.

Contact

Email: support@plantogether.app

Include This in Your Message

  • Device model and iOS/macOS version
  • App version/build number
  • Short description of the issue
  • Screenshots (if available)

Common Issues

  • Sign in with Apple fails: Verify your internet connection and retry. If it persists, include a screenshot of the error.
  • Invites not opening app: Ensure invite links use plantogether://join?code=... and that you have the latest app build.
  • Cloud sync not updating: Open Activity → Cloud Account and tap “Refresh Cloud.”
  • Calendar add failed: Enable Calendar access in iOS Settings for PlanTogether.

Response Time

Typical response time is 1–3 business days.
